Introduction to HTML

HTML (HyperText Markup Language) is the standard language for creating web pages. It uses tags to structure content on the web. This tutorial covers basic HTML commands to help you get started.

Basic HTML Tags

Below is a table of essential HTML tags, their descriptions, and examples.

Tag Description Example
<html> Defines the root of an HTML document <html>...</html>
<head> Contains metadata about the document <head><title>My Page</title></head>
<body> Contains the visible content of the page <body><p>Hello!</p></body>
<h1> to <h6> Defines headings (h1 is largest) <h1>Main Heading</h1>
<p> Defines a paragraph <p>This is a paragraph.</p>
<a> Creates a hyperlink <a href="https://example.com">Link</a>
<img> Embeds an image <img src="image.jpg" alt="Image">
<ul> & <li> Creates an unordered list <ul><li>Item 1</li></ul>

HTML Attributes

Attributes provide additional information about HTML elements. They are always specified in the start tag and usually come in name/value pairs like name="value".

Attribute Description Example
id Specifies a unique identifier for an element <div id="myDiv">Content</div>
class Specifies one or more class names for an element <p class="text">Paragraph</p>
href Specifies the URL for a hyperlink <a href="https://example.com">Link</a>
src Specifies the source file for an image or media <img src="image.jpg">
alt Provides alternative text for images <img src="image.jpg" alt="Description">

Basic Text Formatting Tags

These tags help you style text to make it bold, italic, or emphasized. They are simple and commonly used.

Tag Description Example
<b> Makes text bold <b>Bold Text</b>
<i> Makes text italic <i>Italic Text</i>
<strong> Emphasizes text (usually bold) <strong>Important Text</strong>
<em> Emphasizes text (usually italic) <em>Emphasized Text</em>

Simple HTML Example

Here's a basic HTML code snippet showing some tags in action:

<!DOCTYPE html>
<html>
<head>
    <title>My First Page</title>
</head>
<body>
    <h1>Welcome!</h1>
    <p>This is a <strong>bold</strong> paragraph with a <a href="https://example.com">link</a>.</p>
</body>
</html>
